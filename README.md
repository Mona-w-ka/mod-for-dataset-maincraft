# Minecraft Dataset: Сбор скриншотов для датасета

Этот проект позволяет автоматически собирать скриншоты в Minecraft с помощью Python-скрипта и небольшого Fabric-мода.  
Подходит для создания датасета

> Проект фокусируется на процессе сбора данных, а не на самом моде как конечном продукте.

---

## Что нужно установить

1. **Minecraft** 1.20.6
2. **Fabric Loader** — (https://fabricmc.net/use/)
3. **Python 3.8+** — (https://www.python.org/downloads/)
4. Библиотеки Python:
   pyautogui
## Как запустить?

1. Добавить modid.jar в папку mods в директории Майнкрафта
2. Запустить screen.py
3. Запустить в Майнкрафт, создать новый мир и зайти в него
